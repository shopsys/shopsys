### List of payment methods [/graphql{?payments}]

#### POST [POST]

Returns complete list of payment methods

- Request (application/json)

        {
            query: payments {
                uuid
                name
                description
                instruction
                position
                price {
                    priceWithVat
                    priceWithoutVat
                    vatAmount
                }
                images {
                    type
                    position
                    size
                    url
                    width
                    height
                }
                transports {
                    uuid
                    name
                }
            }
        }
