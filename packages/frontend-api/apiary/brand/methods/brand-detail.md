### Brand detail [/graphql{?brand_detail}]

#### POST [POST]

Returns brand filtered using UUID

- Request (application/json)

    - Attributes

        - uuid (required)

    - Body

            {
                brand(uuid: "7eef3b9e-4e57-4166-b587-d1fff68987c0") {
                    uuid
                    name
                    description
                    link
                    seoTitle
                    seoMetaDescription
                    seoH1
                    products (first: 5) {
                        edges {
                            node {
                                name
                            }
                        }
                    }
                    images{
                        url,
                        type,
                        size,
                        width,
                        height,
                        position
                    }
                }
            }
