### Transport method detail [/graphql{?transport_detail}]

#### POST [POST]

Returns transport filtered using UUID

- Request (application/json)

    - Attributes

        - uuid (required)

    - Body

            {
                query: transport(uuid: "edbafa26-9306-46ed-b1e8-d1570f072832") {
                    uuid
                    name
                    description
                    instruction
                    position
                    price {
                        priceWithVat
                        priceWithoutVat
                        vatAmount
                    }
                    images {
                        type
                        position
                        size
                        url
                        width
                        height
                    }
                    payments {
                        uuid
                        name
                    }
                }
            }
