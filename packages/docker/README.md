# Shopsys Framework Docker

## About
In Shopsys Framework we use Docker as recommended way to install and run the application. 
For the purposes of building a new project use our [shopsys/project-base](https://github.com/shopsys/project-base),
which is fully ready as the base for building your Shopsys Framework project.
For detailed instructions follow [Installation Using Docker](https://github.com/shopsys/shopsys/blob/master/docs/installation/installation-using-docker.md).
